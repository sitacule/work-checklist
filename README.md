# 📋 팀 업무체크리스트

> 팀원 개인 업무 관리부터 팀장 주간보고 · 대표이사 월간보고까지  
> 하나의 웹 페이지에서 자동으로 처리하는 팀 업무관리 시스템

🔗 **배포 주소**: [https://sitacule.github.io/work-checklist/](https://sitacule.github.io/work-checklist/)

---

## 📌 주요 기능

### 👤 팀원
| 기능 | 설명 |
|------|------|
| 내 업무 등록 · 수정 · 삭제 | 본인 업무만 관리 (타 팀원 업무 접근 불가) |
| 우선순위 관리 | A(급함·중요) / B(급함·덜중요) / C(중요·덜급) / D(덜중요·덜급) |
| 처리여부 관리 | 진행중 / 처리완료 / 보류 |
| 우선순위별 현황 | 본인 업무를 A~D 매트릭스로 시각화 |
| 월별 현황 | 본인 업무의 월별 완료·진행·보류 통계 |
| 주간보고 자동 생성 | 해당 주 업무를 자동 정리 → 팀장 제출용 |
| 월간보고 자동 생성 | 해당 월 업무를 자동 정리 → 팀장 제출용 |

### 👑 팀장
| 기능 | 설명 |
|------|------|
| 전체 현황 대시보드 | 팀원별 업무 수 · 완료율 · 기한초과 요약 카드 |
| 전체 업무 목록 | 담당자·우선순위·처리여부 필터 |
| 우선순위별 현황 | 팀 전체 A~D 매트릭스 |
| 월별 현황 | 팀원별 월별 통계 |
| 주간보고 자동 생성 | 전체 팀원 주간 업무 보고서 (팀원별 선택 가능) |
| 월간보고 자동 생성 | 전체 팀원 월간 업무 보고서 → 대표이사 보고용 |

## 🚀 시작하기

### 1단계 — 사이트 접속
```
https://sitacule.github.io/work-checklist/
```

### 2단계 — 초기 설정 (최초 1회)

| 항목 | 설명 |
|------|------|
| **저장 방식** | 로컬 저장(PC 전용) 또는 Firebase 연동(팀 공유) 선택 |
| **팀장 이름** | 팀장 이름 입력 |
| **팀원 목록** | 쉼표로 구분 입력 (예: `홍길동, 김철수, 이영희`) |
| **팀장 비밀번호** | 선택 사항 — 팀장 로그인 보호용 |

### 3단계 — 로그인
- **팀원**: 이름 선택 → 입장
- **팀장**: 👑 팀장 선택 → 비밀번호 입력 → 입장

## ☁️ Firebase 연동 (팀원 실시간 공유)

팀원 전체가 동일한 데이터를 실시간으로 공유하려면 Firebase 설정이 필요합니다.

### 설정 방법 (무료 · 5분 소요)

1. [Firebase 콘솔](https://console.firebase.google.com) 접속
2. **프로젝트 만들기** → 프로젝트명 입력
3. 좌측 메뉴 **Firestore Database** → **데이터베이스 만들기** → **테스트 모드** 선택
4. **프로젝트 설정** (⚙️) → **앱 추가** → 웹(`</>`) 선택
5. 표시되는 설정값에서 아래 3가지 복사:

```js
apiKey: "AIzaSy..."
authDomain: "your-project.firebaseapp.com"
projectId: "your-project-id"
```

6. 초기 설정 화면에서 **Firebase 연동** 선택 후 위 값 입력

> ⚠️ 테스트 모드는 30일 후 보안 규칙 만료됩니다. 장기 사용 시 Firestore 보안 규칙 설정을 권장합니다.

## 📝 보고서 자동 생성

### 주간업무보고 (팀장 보고)

1. `📝 주간보고` 탭 클릭
2. **보고 기준 주** 선택
3. 보고서 자동 생성 확인
4. `🖨 인쇄 / PDF 저장` 클릭 → 브라우저 인쇄창에서 **PDF로 저장**

**자동 생성 내용**
- 이번 주 업무 요약 (완료·진행·보류·기한초과 건수)
- 팀원별: 완료 업무 / 진행중 업무 / 보류 업무 / 다음 주 예정
- 기한 초과 업무 ⚠️ 자동 강조 표시
- 서명란 (팀원 → 팀장)

### 월간업무보고 (대표이사 보고)

1. `📑 월간보고` 탭 클릭
2. **기준 연월** 선택
3. 보고서 자동 생성 확인
4. `🖨 인쇄 / PDF 저장` 클릭

**자동 생성 내용**
- 월간 업무 실적 요약 (총 건수·완료율)
- 팀원별 달성 현황 (완료율 바 포함)
- 우선순위별 달성 현황 (A~D 완료율)
- 주요 완료 업무 목록
- 진행중 업무 현황 / 보류 업무
- 다음 달 업무 계획
- 서명란 (팀장 → 대표이사)
